EduCore Learning Management System

A comprehensive student learning management system developed as my 2nd year final project. Built using React.js for the frontend, Node.js and Express.js for the backend, with PHPMyAdmin for database management. Features include course management, student enrollment, assignment submission, grade tracking, and real-time communication tools. Demonstrates modern full-stack development with responsive design and scalable architecture.

CozyComfort Supply Chain System

A Service-Oriented Component (SOC) architecture project for CozyComfort blanket manufacturing company, developed for 2nd year Service-Oriented Computing using C# and ASP.NET Web Forms. Features SOAP-based Web Services connecting manufacturers, distributors, and sellers with centralized inventory management, order processing, delivery coordination, and reporting. Implements modular services (UserService, ProductService, OrderService) ensuring scalability and maintainability for future business growth.
